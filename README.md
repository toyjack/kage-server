# kage-server

An online font redering server using [Kage-engine](https://github.com/kurgm/kage-engine)

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
